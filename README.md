AvatarGenerator
===============

一个在图片右上角加上红底白字数字的应用，最近在微信朋友圈比较火，但不能自定义头像，用这个可以自定义头像。

<p>
	<span style="font-size:18px"><span style="white-space:pre"></span>最近一段时间在微信朋友圈，在头像的右上角增加一个红底白字的数字，让很多有强迫症的同学点个不停，深深佩服发明这种头像的姑娘，太机智了。但它不能自定义，这是硬伤。。。。</span>
</p>
